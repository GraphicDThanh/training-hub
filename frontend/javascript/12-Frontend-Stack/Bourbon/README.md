## Running ##
> **If you run with vagrant virtual machine**
>
> cd /vagrant => vagrant up => vagrant ssh => cd /vagrant => cd ../../front-end-practices
>
> **Run npm dependencies**

> sudo npm install

> **Run bower to get libraries**

> bower install

> **Run gulp to build project**

> gulp
